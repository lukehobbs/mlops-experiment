# MLOPS-Experiment
## Prerequisites:
 - Install [python](https://www.python.org/) 
    ```
    brew install python@3.9
    ```
 - Install requirements
    ```
    pip install -r requirements.txt
    ```
## The Project
 - Run [Jupyter Lab](https://jupyterlab.readthedocs.io/en/stable/)
    ```
    jupyter-lab
    ```
